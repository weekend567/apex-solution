# Oheneba Art Gallery Ecommerce Website

A premium, responsive ecommerce storefront for **Oheneba Art Gallery**.

## Features

- High-impact hero and gallery-led visual design
- Search and category filtering for artworks
- Shopping cart with quantity controls and live subtotal
- Checkout experience with:
  - **Mobile Money** (MTN, Vodafone Cash, AirtelTigo)
  - **Visa Card**
  - **Mastercard**
- Local and international currency display (GHS / USD)
- Graceful fallback for missing artwork files

## Run locally

Because this is a static site, open `index.html` directly in a browser, or run a local server:

```bash
python3 -m http.server 8080
```

Then visit:

`http://localhost:8080`

## Artwork image placement

Place your artwork files in:

`assets/images/`

Use the filenames listed in:

`assets/images/README.md`
